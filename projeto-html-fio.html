(Voltar ao índice)

"hr/" cria uma linha
"small" "/small" escreve em fonte menor
Algumas tags não precisam de fechamento (ex:imput; img)
Atributos das tags sáo propriedades que a tag pode ter, podendo ser genéricas de todas ou específicas
Genéricas: ex: id (identificador); style=""(no css, color); class(no css, para padronização)
Específicas: imput type="text" ; img src"endereço da imagem"
